# Smart Job Tracker

A full-stack MERN application for tracking job applications.

## Features

- User Registration & Login
- JWT Authentication
- Protected Routes
- Create, Update, Delete Jobs
- Search Jobs
- Priority Management
- Notes Management
- Dashboard Analytics

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcrypt

## Run Locally

### Backend

npm install
npm run dev

### Frontend

npm install
npm run dev
